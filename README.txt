HW6 README

[요구사항]
(a) 물체의 배치 및 움직임의 설정
 i. 용, 고질라가 x 축 상에서 서로 마주보고 서있고, 오토바이가 그 사이에 배치되어있다.
 ii. 사람(ben)과 거미와 늑대를 배치하였다.    
 iii. 오토바이가 용과 고질라 사이를 8자를 그리며 움직이면서 회전한다.
 iv.
   1) 사람이 zy 평면 상에서 원을 그리며 회전하는데 이 때 키가 늘어났다 줄었다 한다.
   2) 늑대는 z축 상에서 움직이는데, 원점에 가까워질수록 커졌다가 다시 작아지며 사라진다.
   3) 거미는 용과 고질라의 머리를 밟고서 무지개를 그리며 왔다갔다한다.

(b) 광원 의 배치
 i. 0번과 1번 광원은 켜져있고 2번과 3번 광원은 꺼져있다.
  키보드의 0,1,2,3을 누르면 각각 0,1,2,3번 광원을 on/off할 수 있다.
 ii. 
  1) 0번 - 연두색 점광원을 세상좌표계에 배치하였다.
  2) 1번 - 연보라색 점광원을 세상좌표계에 배치하였다.
 iii. 2번 광원은 빨간색 스폿광원으로 카메라에 고정되어 카메라가 움직일 때 함께 움직인다.
 iv. 3번 광원은 파란색 스폿광원으로 거미의 아래쪽에 바닥을 비추도록 배치되어있다.
     거미가 움직일 때 마다 3번광원의 position과 direction도 함께 움직인다.

(c) 카메라 움직임 생성
 i.
  1) 키보드의 좌우 방향키로 u축 방향으로 이동(translation)할 수 있다.
  2) 키보드의 상하 방향키로 v축 방향으로 이동(translation)할 수 있다.
  3) 마우스를 우클릭한 채로 위아래로 움직이면 n축 방향으로 이동(translation)할 수 있다.
 ii.
  1) 마우스를 왼클릭한 채로 좌우로 움직이면 u축 둘레로 회전(rotation)할 수 있다.
  2) 마우스를 왼클릭한 채로 상하로 움직이면 v축 둘레로 회전(rotation)할 수 있다.
  3) 마우스를 우클릭한 채로 좌우로 움직이면 n축 둘레로 회전(rotation)할 수 있다.
 iii. 키보드의 +/- 키를 누르면 zoom in/zoom out할 수 있다.

(d) 새로운 텍스쳐 적용
 i. 키보드의 't'키를 누르면 용의 텍스쳐가 번갈아 가면서 바뀐다.
   초기상태로는 원래의 텍스쳐(호랑이)가 입혀져있고
   't'를 누르면 반려묘 루나의 사진이 텍스쳐로 입혀진다.

(e) 쉐이더
 i. 키보드의 'q'키를 누르면 고질라에게 Gouraud shading과 Phong shading이 번갈아 적용된다.
   초기에는 phong shading이 적용되어있다. 조명 2번을 비춘상태에서 확인하면 더 명확하게 드러난다.
 ii. 키보드의 'w'키를 누르면 늑대에 적용되는 광원효과가 토글된다.
   먼저 스폿광원(2번)에 대해 블라인드 효과가 적용되고
   시간이 흐름에따라 반사하는 색의 rgb 영역값이 계속 변화하여 늑대의 색이 변한다.
   0,1번 광원을 끄고 2번 광원을 늑대에게 비추면 더 명확하게 드러난다.
 iii. 키보드의 'e'키를 누르면 사람(ben)에게 원형의 구멍이 생기고 e를 계속 누르면 구멍이 더 많아진다.
